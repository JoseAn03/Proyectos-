# 📖 Guías y Documentación

Guías paso a paso para usar herramientas de IA en la generación de reportes operacionales.

## Guía disponible

### 🤖 Cómo usar Claude para el Reporte de Reservas

Guía completa para generar el reporte Game Plan usando Claude (Anthropic) como alternativa a las macros VBA.

**Incluye:**
- Cómo subir el archivo Excel a Claude
- El prompt exacto para procesar el manifiesto
- Especificación de columnas, marcas e indicadores
- Explicación de símbolos (⭐FL⭐, MAIN VIP, EXPEDIA)
- Preguntas frecuentes

**Ventaja:** No requiere instalar nada — funciona desde el navegador en cualquier dispositivo.
